## Goal

Start with the smallest possible in-browser Python example that should work: **generate points and plot them**.  
Once this renders reliably in your browser runner, we can add the gradient descent loop next.

---

## Minimal Python snippet: generate and plot points

Copy/paste this into your in-browser Python runner and click **Run**:

```python
import numpy as np
import matplotlib.pyplot as plt

# reproducible random points
rng = np.random.default_rng(0)
n = 200
x = rng.uniform(0, 50, size=n)
y = 0.35 * x + 7.0 + rng.normal(0, 3.0, size=n)

plt.figure()
plt.scatter(x, y, alpha=0.35)
plt.title("Synthetic points")
plt.xlabel("x")
plt.ylabel("y")
plt.show()
