## Minimal interactive cell

If this cell works on your site, your Pyodide integration is functioning. Then we can add gradient descent on top.

<PyodideRunner
  id="gd_scatter_demo"
  title="Generate + plot synthetic points"
  height={240}
  autoRun={true}
  initialCode={`import numpy as np
import matplotlib.pyplot as plt

rng = np.random.default_rng(0)
n = 200
x = rng.uniform(0, 50, size=n)
y = 0.35 * x + 7.0 + rng.normal(0, 3.0, size=n)

plt.figure()
plt.scatter(x, y, alpha=0.35)
plt.title("Synthetic points")
plt.xlabel("x")
plt.ylabel("y")
plt.show()
`}
/>
